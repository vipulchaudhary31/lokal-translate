# Plugin Assets

## Logo for Community Publishing

Place your plugin icon here for use when publishing to Figma Community:

- **Filename:** `icon.png` (or `icon.svg`)
- **Size:** 512×512 px recommended
- **Format:** PNG or SVG (transparent background recommended)

Figma will prompt you to upload the icon during the Community publishing flow. The icon in this folder will not appear during local development—Figma uses a default icon for unpublished plugins.
